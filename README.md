# ley-29090-motor-decision-v4

MVP v0.4: Motor de decisión + generador de checklist (módulos).

## Estructura
- `/docs/index.html` UI (diferenciada por clasificación)
- `/docs/app/engine.js` motor (evalúa reglas)
- `/docs/app/checklist.js` compone checklist por módulos
- `/docs/data/reglas_29090.json` reglas del motor
- `/docs/data/checklists_29090.json` módulos de checklist (validables)
- `/docs/data/casos_prueba.json` casos de prueba

## GitHub Pages
Settings → Pages → Deploy from a branch → main → /docs

## Nota
La base de checklist es modular y está pensada para validación y ajuste del usuario según TUO 29090, reglamentos y TUPA municipal aplicable.
